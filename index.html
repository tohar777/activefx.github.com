What's ActiveFX?:


ActiveFX is a open source projects that provides a set tools made for The Link Programming language to run on the web. ActiveFX provides:DOM Manipulation(page interactions),AFXGL(Graphics Library using SDL and OGL) and AFXAudio


ActiveFX is also providing a heavy modified dialect of Link named LinkScript which is similar to languages like lua.
